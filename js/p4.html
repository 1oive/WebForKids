Циклы в JavaScript позволяют выполнять одни и те же действия многократно, не повторяя код. Это особенно полезно, когда нужно обработать большое количество данных или выполнить действие определённое количество раз. В этом уроке мы познакомимся с циклом for.

Цикл for используется для выполнения блока кода указанное количество раз. Он состоит из трёх частей: инициализация (тут задается переменная-счетчик), условие (при котором цикл будет работать) и изменение значения переменной-счетчика на каждом новом шаге.


Пример: for (let i = 1; i <= 10; i++) {
console.log(i);
}
