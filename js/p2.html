let num1 = 20; // Первое число
let num2 = 25; // Второе число
// Складываем переменные, сохраняем в переменную
let sum = num1 + num2;
// Выводим результат на экран
document.write("Сумма: " + sum);

Переменные в JavaScript — это контейнеры для хранения данных. С их помощью можно сохранять значения (например, числа или строки) и использовать их позже в программе. Операции с переменными позволяют выполнять вычисления, такие как сложение, вычитание, умножение и деление. Это основа любого программирования, и именно с этого мы начнем изучение JavaScript.

Переменная — это место в памяти компьютера, где можно хранить значение. Перед использованием переменной её нужно объявить с помощью ключевых слов let, const или var.


• let: Используется для объявления переменной, значение которой может меняться.

• const: Используется для объявления константы, значение которой нельзя изменить после присвоения.

• var: Устаревший способ объявления переменных, но всё ещё работает.

Основные математические действия: сложение (+), вычитание (-), умножение (*), деление (/).
