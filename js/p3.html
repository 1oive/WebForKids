let number = 11;
if (number % 2 === 0) {
document.write("Число четное");
} else {
document.write("Число нечетное");
}

Условные операторы в JavaScript позволяют выполнять разные действия в зависимости от значения условия. Например, с их помощью можно проверить, является ли число четным или нечетным. Для этого используется оператор % (остаток от деления), который помогает определить, делится ли число на другое без остатка.

Условный оператор if{...}else{...} позволяет выполнять разные блоки кода в зависимости от того, истинно (true) или ложно (false) условие.


Оператор % возвращает остаток от деления одного числа на другое. Если число делится на 2 без остатка (число % 2 === 0), то оно является четным.
