Формы в HTML позволяют создавать интерактивные элементы на сайте, с помощью которых пользователи могут отправлять данные. Это могут быть текстовые поля для ввода информации, выпадающие списки, флажки (checkbox), кнопки и многое другое. Формы широко используются для регистрации, опросов, заказов и других целей.

• Тег (<form>): Обозначает форму. Все её элементы помещаются внутрь этого тега.
• Атрибут (action): Указывает URL-адрес, куда будут отправлены данные формы после её отправки.
• Атрибут (method): Определяет способ отправки данных (GET или POST).
Элементы форм:
• Текстовое поле (<input type="text">): Для ввода текста.
• Чекбокс (<input type="checkbox">): Для выбора нескольких вариантов.
• Радио-кнопка (<input type="radio">): Для выбора одного варианта из нескольких.
• Кнопка отправки (<button type="submit">): Для отправки данных формы.
• Метка (<label>): Добавляет описание к элементу формы, делая его более понятным для пользователя.
