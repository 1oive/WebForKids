Списки в HTML позволяют структурировать информацию, делая её более читабельной и понятной. Существует два основных типа списков: неупорядоченные (bullet points) и упорядоченные (нумерованные). Используя эти списки, можно создавать меню, инструкции, списки покупок и многое другое.

• Неупорядоченный список (<li>):
Используется для создания списка элементов без конкретного порядка. Каждый элемент списка помечается тегом <li> (list item).

• Упорядоченный список (<ol>):
Используется для создания нумерованного списка, где порядок элементов имеет значение. Также каждый элемент списка обозначается тегом <li>.
