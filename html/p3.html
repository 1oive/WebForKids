Для создания интерактивных элементов на веб-странице, таких как ссылки и изображения, HTML предоставляет специальные теги: <a> для ссылок и <img> для картинок. С их помощью можно добавить на страницу возможность перехода на другие сайты или разделы вашего сайта, а также размещать изображения.

Тег <a> (ссылка): Используется для создания гиперссылок. Атрибут href указывает URL-адрес, куда нужно перейти при клике на ссылку.
Пример: <a href="https://example.com">Перейти на Example</a>

Тег <img> (картинка): Используется для добавления изображений на страницу. Атрибут src указывает путь к файлу картинки, а alt — текстовое описание изображения (важно для доступности).
Пример: <img src="https://example.com/image.jpg" alt="Описание картинки">

Задание 1

Добавьте на свою страницу следующие элементы:

Ссылку на ваш любимый сайт (например, сайт с играми или образовательный ресурс). Укажите название ссылки, чтобы было понятно, куда она ведет.
Картинку, которая вам нравится. Вы можете найти картинку в интернете и скопировать её прямую ссылку.
