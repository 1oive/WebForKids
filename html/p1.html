HTML (HyperText Markup Language) — это язык разметки, с помощью которого можно создавать структуру веб-страниц. С его помощью вы сможете описать заголовки, параграфы, изображения, формы и другие элементы, которые будут отображаться в браузере.

Задание 1

Вставить код-пример в поле справа:


<h1>Добро пожаловать на мой сайт!</h1>
<h2>Здесь вы найдете интересные вещи</h2>

<form>
    <!-- Поле для ввода имени -->
    <label for="name">Ваше имя:</label><br>
    <input type="text" id="name" name="name"><br><br>

    <!-- Чекбокс -->
    <label for="subscribe">Хотите получать новости?</label>
    <input type="checkbox" id="subscribe" name="subscribe"><br><br>

    <!-- Кнопка отправки -->
    <button >Отправить</button>
</form>
