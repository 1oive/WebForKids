HTML позволяет создавать заголовки разных уровней с помощью специальных тегов <h1> до <h6>. Каждый уровень отражает важность заголовка: <h1> используется для главного заголовка страницы, а <h6> — для наименее значимых подзаголовков. Также для обычного текста применяется тег <p>, который обозначает параграф или абзац.

Задание 2

Создайте структуру текста, используя заголовки разных уровней и несколько абзацев. Отразите следующую иерархию:
Основной заголовок (<h1>): "Моя учеба".
1.1. Подзаголовок первого уровня (<h2>): "Что я изучаю".
1.1.1. Подзаголовок второго уровня (<h3>): "Программирование".
1.2. Подзаголовок первого уровня (<h2>): "Мои хобби".
1.2.1. Подзаголовок второго уровня (<h3>): "Рисование".
1.2.2. Подзаголовок второго уровня (<h3>): "Музыка".

После каждого подзаголовка добавьте параграф (<p>) с кратким описанием (например, "Я люблю изучать математику, потому что она помогает мне решать сложные задачи").
