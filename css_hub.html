Уроки

Урок 1. Знакомство

Познакомимся с основами CSS и создадим простой стиль для элемента, чтобы увидеть, как работает язык стилей.

Урок 2. Стилизация шрифтов

Научимся изменять внешний вид текста: размер, цвет, семейство шрифтов, а также добавлять курсив или жирность.

Урок 3. Отступы. Margin, padding

Изучим отступы между элементами (margin) и внутренние отступы внутри элементов (padding), что поможет лучше организовать макет страницы.

Урок 4. Стилизация списков

Разберемся как менять маркеры списков, включая использование изображений вместо стандартных точек или номеров.

Урок 5. Позиционирование

Погрузимся в позиционирование элементов на странице с помощью свойств position, научимся точно размещать элементы относительно друг друга или всего документа.

Урок 6. Адаптивные запросы

Узнаем про медиа-запросы, которые позволяют изменять стили в зависимости от размера экрана устройства пользователя.

Урок 7. Flexbox

Освоим технологию Flexbox для создания гибких и адаптивных макетов, где элементы автоматически распределяются по строкам или столбцам.

Урок 8. Grid

Изучим CSS Grid Layout — мощный инструмент для создания двумерных сеток, который позволяет легко контролировать расположение элементов как по горизонтали, так и по вертикали.

Урок 9. Переменные

Познакомимся с CSS-переменными, которые помогают хранить значения стилей в одном месте и использовать их по всему проекту, делая код более организованным.

Урок 10. Анимации

Научимся создавать динамические эффекты с помощью CSS-анимаций, включая переходы (transitions) и ключевые кадры (keyframes).
