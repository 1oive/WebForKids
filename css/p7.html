.container-test {
display: flex; /* Активируем Flexbox */
flex-direction: row; /* Располагаем элементы в строку */
gap: 20px; /* Добавляем пространство между элементами */
}

Flexbox — это мощный инструмент CSS, который позволяет создавать гибкие и адаптивные макеты. С его помощью можно легко выравнивать элементы, располагать их в строку или столбец, а также контролировать пространство между ними. Flexbox особенно полезен для создания современных интерфейсов, где важна точная организация контента.

Контейнер - это родительский элемент, которому мы применяем display: flex;. Все дочерние элементы внутри него становятся "элементами Flex".
Элементы Flex - это дочерние элементы внутри контейнера. Их расположение и поведение зависят от настроек контейнера.

Основные свойства:
flex-direction:
- row: элементы располагаются в строку (слева направо, по умолчанию).
- column: элементы располагаются в столбец (сверху вниз).
gap:
- Задаёт пространство между элементами Flexbox.
