Списки — важный элемент веб-страниц, который помогает структурировать информацию. С помощью CSS можно не только менять цвет и размер шрифта списка, но и полностью изменять внешний вид маркеров (точек или номеров) у элементов списка. Например, вы можете заменить стандартные маркеры на изображения, что сделает список более интересным и уникальным.

1. list-style-type определяет тип маркера для элементов списка. Возможные значения: circle, square, none...
2. list-style-image позволяет использовать изображение в качестве маркера списка. Значение указывается как путь к изображению.
2. list-style-position определяет положение маркера относительно текста. Возможные значения: inside, outside.
