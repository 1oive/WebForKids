Тег <p> используется для создания абзацев текста на веб-странице. С помощью CSS можно стилизовать этот текст, меняя его внешний вид: размер шрифта, цвет, стиль (жирный, курсив), а также фон за текстом. Это позволяет сделать текст более привлекательным и удобным для чтения.

Основные свойства для стилизации текста:
1. font-family: Указывает название шрифта. Например: 'Arial', 'Times New Roman'.
2. font-size: Задаёт размер шрифта. Можно указывать в пикселях (px), процентах или других единицах.
3. font-style: Определяет стиль шрифта. Возможные значения: normal, italic, oblique.
4. font-weight: Задаёт толщину шрифта. Например: normal, bold, lighter или числовые значения от 100 до 900.
5. color: Определяет цвет текста. Можно использовать названия цветов, HEX-коды или RGB.
6. background-color: Задаёт цвет фона за текстом
7. text-align: Выравнивает текст внутри элемента. Возможные значения: left, center, right, justify.
