Позиционирование в CSS позволяет точно контролировать расположение элементов на странице. Это особенно полезно, когда нужно разместить элементы в определённых местах, например, в углу экрана или относительно других элементов. Существует несколько значений свойства position, но мы рассмотрим два основных: relative и absolute.

1. position: relative: Элемент позиционируется относительно своего обычного положения в потоке документа. Изменяя свойства top, right, bottom, left, можно сдвинуть элемент от его первоначального места.
2. position: absolute: Элемент позиционируется относительно ближайшего родительского элемента с position: relative. Также используются свойства top, right, bottom, left для точного позиционирования.

На экране уже есть <div> .box, который необходимо переместить в правый верхний угол страницы. Для этого используйте свойство position: absolute; и соответствующие значения для top и right.
