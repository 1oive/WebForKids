:root {
--triangle-color: green; /* Цвет треугольника */
--triangle-width: 200px; /* Ширина треугольника */
}

.triangle {
width: var(--triangle-width); /* Используем переменную для ширины */
background-color: var(--triangle-color);
}

CSS-переменные — это мощный инструмент, который позволяет хранить и переиспользовать значения (например, цвета, размеры или отступы) в одном месте, а затем использовать их по всему проекту. Это делает код более организованным, легким для поддержки и удобным для быстрых изменений. Переменные особенно полезны, когда одно значение используется много раз на странице.

CSS-переменные (или кастомные свойства) позволяют определить значение один раз и использовать его многократно. Они начинаются с двух дефисов (--) и объявляются внутри блока :root или конкретного элемента.

На экране уже есть треугольник .triangle. Ваша задача — задать две переменные:

- Для ширины (width).
