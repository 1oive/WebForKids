CSS (Cascading Style Sheets) — это язык стилей, который используется для оформления веб-страниц, созданных с помощью HTML. Если HTML определяет структуру страницы, то CSS отвечает за её внешний вид: цвета, шрифты, размеры и расположение элементов.

Для применения CSS можно использовать три способа:
1. Внутренний стиль (style атрибут внутри тега).
2. <style> блок в HTML-документе.
3. Отдельный файл .css, подключаемый через <link>.

Один из важных способов работы с CSS — использование классов. Класс позволяет выбрать конкретный элемент или группу элементов на странице и применить к ним определённые стили. Для этого используется точка перед названием класса: .classname.

Задание 1

На экране уже есть треугольник, созданный с помощью CSS и HTML. Ваша задача — обратиться к нему через класс .triangle и изменить его фоновый цвет на зелёный (green).

.triangle {
background-color: <Название цвета>;
}
