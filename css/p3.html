.box1 {
margin-left: 10px;
margin-bottom: 20px; /* Добавленный отступ */ }
.box2 {
padding: 20px; /* Добавленное внутренний отступ */ }

Отступы в CSS позволяют создавать пространство между элементами на странице, делая дизайн более airy (воздушным) и организованным. Существует два основных типа отступов: margin и padding. Они помогают контролировать расстояние между элементами и их содержимым.

1. Margin добавляет пространство снаружи элемента, не увеличивая его сам.
2. Padding добавляет пространство внутри элемента, увеличивая его размер.
Можно задавать как одно значение для всех сторон (padding: 10px;), так и отдельно (padding-bottom: 10px;)

Задание 1

На экране уже есть два квадрата (.box1, .box2), расположенные друг над другом. Ваша задача — использовать свойства margin и padding, чтобы: 1.Увеличить расстояние между двумя квадратами (используйте margin). 2.Добавить внутреннее пространство внутри второго квадрата, чтобы текст внутри него стал дальше от границы (используйте padding).
