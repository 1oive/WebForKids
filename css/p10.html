Анимации на чистом CSS позволяют создавать динамические и интерактивные элементы на веб-страницах без использования JavaScript. Это может быть плавное изменение цвета, перемещение объектов, вращение или другие эффекты. CSS предоставляет два основных способа создания анимаций: transition (переходы) и animation (ключевые кадры). В этом уроке мы рассмотрим оба подхода.

- transition позволяет создавать плавные изменения между состояниями элемента, например, при наведении мыши (hover) или клике.
- animation позволяет создавать более сложные анимации с помощью ключевых кадров (@keyframes). Каждый ключевой кадр определяет состояние элемента в определённый момент времени.
