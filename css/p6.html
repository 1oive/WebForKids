Адаптивный дизайн позволяет веб-сайтам выглядеть одинаково хорошо на устройствах с разными размерами экрана, будь то десктоп, планшет или мобильный телефон. Для этого используется технология медиа-запросов (media queries), которая позволяет применять определённые стили только при выполнении конкретных условий, таких как ширина экрана.

Медиа-запросы — это инструмент CSS, который позволяет применять стили только тогда, когда выполняются определённые условия (например, ширина экрана меньше или больше определённого значения).
Примеры:
@media (max-width: 1000px) — применяется, если ширина экрана меньше или равна 1000 пикселей.
@media (min-width: 1000px) — применяется, если ширина экрана больше или равна 1000 пикселей.

На экране уже есть <div> с классом .adaptive-box, который имеет определённые стили для десктопной версии сайта. Ваша задача — создать медиа-запрос для экранов шириной более 1000px и изменить следующие свойства элемента: увеличить ширину до 400px; Изменить цвет фона на зелёный (green).
