Grid (CSS Grid Layout) — это мощный инструмент для создания сложных и гибких макетов веб-страниц. Он позволяет организовать элементы в двумерной сетке, где можно контролировать как строки, так и столбцы. Grid особенно полезен для создания адаптивных макетов, где важно точно размещать элементы на странице.

Контейнер - это родительский элемент, которому мы применяем display: grid;. Все дочерние элементы внутри него становятся "элементами Grid".
Элементы Grid - дочерние элементы внутри контейнера. Их расположение и поведение зависят от настроек контейнера.

Основные свойства:
grid-template-columns:
- grid-template-columns: 1fr 1fr; — создаст два равных столбца.
- grid-template-columns: 200px 1fr; — первый столбец будет иметь фиксированную ширину 200px, второй займет оставшееся пространство.
grid-template-rows:
- Определяет количество и размер строк в сетке. Работает аналогично grid-template-columns.
gap:
- Задаёт пространство между элементами сетки. Можно использовать row-gap (между строками) и column-gap (между столбцами).
